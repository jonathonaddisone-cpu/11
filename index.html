搭建路线

推荐顺序

  1. 安装 VS Code 或 Cursor 作为主编辑器。
  2. 安装 Git，配置用户名和邮箱。
  3. 安装 Node.js LTS 与 Python，用于运行大多数项目。
  4. 启用中文界面、中文提示词模板和常用扩展。
  5. 把第一个作品发布到社区，接受反馈后迭代。
